About UgTalk

Project Overview

UgTalk is an AI-Powered Web Service built with Flask that translates English text into multiple Ugandan languages using machine learning.

The system uses TF-IDF vectorization and Nearest Neighbors to find the best translations based on the training dataset.

📊 Dataset: Language_translation_models_dataset.csv - 2,000+ records from 7 Ugandan languages

Technology Stack

  • Backend: Flask (Python)
  • AI/ML: Scikit-learn (TF-IDF, k-NN)
  • Frontend: Bootstrap 5, Chart.js
  • Database: Session-based history

Supported Languages

  • 🇺🇬 Luganda
  • 🇺🇬 Lusoga
  • 🇺🇬 Acholi
  • 🇺🇬 Runyankole
  • 🇺🇬 Ateso
  • 🇺🇬 Lugbara
  • 🇺🇬 Rukiga
